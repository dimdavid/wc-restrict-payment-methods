Restricts Woocommerce payment methods by individual products. If the product is in the cart, the method will not be shown.
Restringe métodos de pagamento do Woocommerce por produto individualmente. Se o produto estiver no carrinho, o méthodo não estará disponível.

== Description ==

### What is? ###

This plugin add in the product admin page a new tab to set the restricts payment methods.

= Features =

* Add new tab in the product admin page;
* If method is selected, it will not be shown in checkout page.

### O que é? ###

Este plugin adiciona uma nova aba na página de edição do produto para restringir métodos de pagamento.

= Funcionalidades =

* Adiciona nova aba na página de edição do produto;
* Se o método for selecionado, não será mostrado na página finalizar-compra.

== Installation ==

* Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer;
* Activate the plugin;
* Find the "Restrict Payment Methods" tab in you product admin page.

### Instalação (em portugês) ###

* Envie os arquivos deste plugin para sua pasta de plugins ou instale utilizando o instalador de plugins do WordPress;
* Ative o plugin.
* Encontre a aba "Restrict Payment Methods" na página de edição do produto.
